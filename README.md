# Customer Churn Analysis and Prediction

## Overview

This project explores customer churn in a telecommunications dataset and builds machine learning models to predict whether a customer is likely to leave the service.

## Objectives

* What customer churn is and contributing factors to it
* Perform data cleaning and exploratory data analysis
* Visualize customer behavior and churn patterns
* Train and evaluate predictive machine learning models
* Interpret model predictions using SHAP

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly
* Scikit-Learn
* XGBoost
* SHAP

## Dataset

The project uses the Telco Customer Churn dataset containing customer demographics, account information, service subscriptions, and churn status

## Project Structure

* `ProjectJournal.ipynb` – Main notebook containing analysis and modeling
* `requirements.txt` – Project dependencies
* `Data/` – Dataset file

## Results

The notebook includes exploratory analysis, feature engineering, model training, evaluation metrics, and model interpretation

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Open the Jupyter notebook and run all cells:

```bash
jupyter notebook ProjectJournal.ipynb
```
